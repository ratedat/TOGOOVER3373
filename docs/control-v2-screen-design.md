# Control-v2 Screen Design

## 目的

`/control-v2` は現在の主操作画面です。旧 `/control` は互換URLとしてControl v2へ解決し、編集導線は「共通設定」「オペレーター」「秘宝」「特殊値」「OBS設定」「サイドカー」に集約します。

## 1360x920 一画面ワイヤー

```text
+--------------------------------------------------------------------------------+
| IS# / Campaign title                         Sidecar | Overlay | Save |
+---------------------------+-------------------------------+--------------------+
| ラン設定                  | オペレーター                  | 秘宝               |
| - 統合戦略                | filters: 実装状態 レア度      | search             |
| - 等級 / 難易度           |          職業 職分            | category columns   |
| - 分隊                    |          並び順 列数          | clear              |
| - ランダム分隊効果        |                               |                    |
| - 演目                    | operator card grid            | relic card grid    |
|                           |                               |                    |
| サマリー                  | click card to select          | click card select  |
| - 秘宝/招集/等級          |                               |                    |
| - 分隊効果                |                               |                    |
| - テンプレート由来        |                               |                    |
+---------------------------+-------------------------------+--------------------+
```

## 領域別の役割

- Topbar: 現在の統合戦略、等級、分隊、保存状態、旧画面への退避リンクを出す。
- Left: ラン設定と自動テンプレート由来の簡易サマリーを出す。特殊値は初回対象外。
- Center: 招集オペレーターを選ぶ。検索入力は使わず、選択式フィルターとカードクリックで操作する。
- Right: 所持秘宝を選ぶ。日本語入力が必要な秘宝検索だけは既存のIME対応部分再描画を維持する。

## 後続移植順

1. ボスフラグと発動効果サマリーを右下または下段へ追加する。
2. OBS URL/分割パーツ設定を別パネルとして追加する。
3. 大会入力レビューとJSON入出力をスタッフ向けモードとして統合する。
4. ADB/OCR候補レビューをサイドカーと共通化する。
